# Singleton Design Pattern

## About the Singleton Design Pattern

Singletons create one-of-a-kind objects that have only one instance, ever. They are the easiest pattern to implement, but the reasons for using a singleton are nuanced and buck most OOP conventions.

## Additional Examples

- Reddit sort algorithm: New, Top, Hot, Controversial
- eCommerce payment handler: Paypal, Venmo, Apple Cash
- Travel Navigation: Car, Walk, Public Transit, Uber
